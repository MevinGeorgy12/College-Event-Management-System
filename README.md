# College Event Management System

A Java Swing and MySQL based application for managing college events, workshops and student registrations.

## Features
- Student Login
- Event Registration
- Workshop Registration
- Admin Dashboard
- Notifications
- Event Management

- ## Database Setup
1. Create database `event_management_db`
2. Import `database_setup.sql`
3. Run the project in NetBeans

## Technologies Used
- Java Swing
- MySQL
- JDBC
- NetBeans IDE
